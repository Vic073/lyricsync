About

LyricSync Pro v1.0
Lyrics sourced from lrclib.net — a free, open, no-auth public lyrics API.
Metadata read/written using mutagen.

Synced lyrics are stored in the SYLT ID3v2 frame.
Plain lyrics are stored in the USLT ID3v2 frame.
Only these two frames are modified — all other tags are preserved.
